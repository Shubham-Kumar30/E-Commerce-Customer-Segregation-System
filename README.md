# E-Commerce-Customer-Segregation-System
This is unsupervised Machine Learning project in which I divided e-commerce customers in different clusters using K-Means Clustering, Agglomerative and DBSCAN clustering. 
Performed Pre-processing - EDA, Feature Engineering and determined optimal K value using Elbow method and Silhouette Score, followed by cluster
characterization and summary profiling to enable targeted marketing strategies.

Conclusions:

1. Cluster 0 (Family Shoppers): ------> Discounts and Coupons offer
   
  Low Income, Low Spending ; 
  More Children, Live with Partners ; 
  High web visits, Low catalog, store purchases ; 
  Poor Response. 

2. Cluster 1 (Important Customers): ------> Loyalty programs (High Purchase, High Benefits)
   
  High Income, High Spending ; 
  Less Children, Live with Partners ; 
  Low web visits, High catalog, store purchases ; 
  Avg Response. 

3. Cluster 2 (Digital Browsers): ------> Heavy Discounts offer
   
  Low Income, Low Spending ; 
  More Children, Live Alone ; 
  High web visits, Low catalog, store purchases ; 
  Avg Response. 

4. Cluster 3 (Premium Customers): ------> offer Personalized and Premium services & also encourage them to use online websites through added benefits
   
  High Income, High Spending ; 
  Less Children, Live Alone ; 
  Low web visits, High catalog, store purchases ; 
  Best Response. 

Finally, Company should Prioritise: Cluster 3 > Cluster 1 > Cluster 0 > Cluster 2

Cluster 3 & 1 ---> Golden Customers

Cluster 0 & 2 ---> Price-sensitive Customers
